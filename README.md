# Tree-Based Chatbot (Python)

A simple data-driven chatbot using a tree structure.

## Features
- Tree-based conversation flow
- Minimal conditional logic
- Data-driven design
- Easy to extend by editing the story tree

## Why this project
Built to practice:
- OOP fundamentals
- State management
- Clean architecture
- Efficient control flow

## How it works
- The chatbot stores the current node
- User input is matched against available choices
- The next node is resolved from data, not conditionals
